# Voodu

> A collaborative Kanban tool to help teams track their work effectively.

### Requirements

- PHP 7.2+
- MySQL 5.7+
- NPM 5.6+
- Composer 1.6.3+

### Setup

1. Create an `.env` file

2. Install Node packages: `yarn install`

3. Install Composer packages: `composer install`

4. Run database migrations: `php artisan migrate`

5. Initialize [Passport](https://laravel.com/docs/5.5/passport): `php artisan passport:install`

6. Run Mix: `yarn dev`
